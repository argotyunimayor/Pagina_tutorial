LINEA HORIZONTAL

Existen dos formas para crear una línea separadora horizontal y una única forma para crear una línea separadora vertical. Para crear una línea horizontal la forma fácil y rápida es utilizar la etiqueta <.hr>, que no se cierra y que crea una línea horizontal simple.

Bordes Simples

'border', que utiliza el siguiente formato: border: 2px solid grey; En este ejemplo, el valor '2px establece el grosor del borde en 2 píxeles, el valor 'solid' especifica el tipo de borde, mientras que finalmente se indica el color, ya sea utilizando el nombre de color, o bien el código de color en hexadecimal (anteponiendo el símbolo #).
